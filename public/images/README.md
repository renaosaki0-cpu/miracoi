# /public/images — 写真の配置ガイド

**このフォルダに画像を置くだけで、サイト全体に自動反映されます。**

## 主要画像

| ファイル | 用途 |
|---------|------|
| `miracoi-icon.png` | ヘッダー・フッター・favicon・apple-touch-icon |
| `hero.jpg` | ファーストビュー |
| `story.jpg` | Storyセクション |
| `cta-bg.jpg` | 最終CTA背景 |
| `og-image.jpg` | OGP（1200×630） |

## プロジェクト

| ファイル | 用途 |
|---------|------|
| `project1.jpg` | プロジェクトページ（交互表示） |
| `project2.jpg` | プロジェクトページ（交互表示） |

## ギャラリー

`gallery1.jpg` 〜 `gallery4.jpg` **のみ**が Gallery に表示されます（この4枚以外は自動配置されません）。

| ファイル | 用途 |
|---------|------|
| `gallery1.jpg` | Gallery 1枚目 |
| `gallery2.jpg` | Gallery 2枚目 |
| `gallery3.jpg` | Gallery 3枚目 |
| `gallery4.jpg` | Gallery 4枚目 |

Gallery 用の画像が1枚もない場合のみ、プレースホルダーが表示されます。

## リターン

| ファイル | 用途 |
|---------|------|
| `return-3000.png` | 3,000円 tier |
| `return-5000.png` | 5,000円 tier |
| `return-10000.png` | 10,000円 tier |
| `return-30000.png` | 30,000円 tier |
| `return-50000.png` | 50,000円 tier |
| `return-100000.png` | 100,000円 tier |
| `return-140000.png` | 140,000円 tier |

`return-*` の画像はリターン一覧専用です。Gallery には使用されません。

## 応援メッセージ

役職順に並んだ順番で `supporter1.jpg`, `supporter2.jpg` … と対応します。

| ファイル | 例 |
|---------|-----|
| `supporter1.jpg` | 1番目（代表理事など上位） |
| `supporter2.jpg` | 2番目 |
| … | 追加時は番号を増やす |

**差し替え方法：** 同じファイル名で上書きするだけです。  
**未配置時：** 美しいプレースホルダーが表示され、エラーにはなりません。
